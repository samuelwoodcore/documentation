# Resolve Account Name

This endpoint allows you to create a transfer recipient and confirm the details of such recipient.

### Request Parameters

| **Parameter** | **Description** |
| --- | --- |
| accountNo | This describes the account number of the recipient. |
| bank_id | This describes the bank`id` of the recipient's bank. |

## Parameters

- **accountNumber** (query, string, required: False): Example: `0051762787`
- **bankCode** (query, string, required: False): Example: `2644245`
