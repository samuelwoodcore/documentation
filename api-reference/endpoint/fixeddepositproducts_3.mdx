# Retrieve a Fixed Deposit Product

Allows retrieval of a fixed deposit product by its Identifier.

##### Path Parameters

| **Attribute** | **Type** | **Description** |
| --- | --- | --- |
| fixedDepositId | String | This describes the `id` of the fixed deposit product. |
