# Download Virtual Nubans

This endpoint is used to download a record of Virtual Nubans created within a period of time. The time period is seperated by the `start` and `end` datetime query paramters.

## Parameters

- **start_date** (query, string, required: False): Example: `2010-01-01`
- **end_date** (query, string, required: False): Example: `2022-08-08`
